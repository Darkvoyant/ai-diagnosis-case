# AI Diagnosis Case

Данный проект предназначен для решения IT-кейса "Искусственный интеллект против 
внеплановых простоев: разработка интеллектуальной системы диагностики". 

## Getting Started

Для работы Вам понадобится Python версии 3.12. Пакетные менеджеры можно ставить
на свой вкус. 

Для примера будет описана установка пакетного менеджера uv.

### Установка пакетного менеджера Python uv

Для установки данного пакетного менеджера Вам необходимо проследовать по 
следующей [инструкции](https://docs.astral.sh/uv/getting-started/installation/). 
Он автоматически поставит Python на Вашу систему, так что ничего дополнительного 
ставить будет не нужно. 

Для установки всех необходимых пакетов и настройки виртуальной среды необходимо
ввести команду:
```
uv sync
```
Для активации виртуальной среды необходимо прописать на macOS и Linux необходимо 
прописать: 
```
source .venv/bin/activate
```
В терминале Windows: 
```
.venv\Scripts\activate
```
Более подробную инструкцию можно найти [здесь](https://docs.astral.sh/uv/pip/environments/#__tabbed_1_2).

Для добавления пакета нужно прописать: 
```
uv add <название пакета>
```

В случае изменений (добавления или обновления пакетов) необходимо также пушить uv.lock.

Для формирования файла requirements.txt можно прописать
```
uv pip freeze > requirements.txts
```
